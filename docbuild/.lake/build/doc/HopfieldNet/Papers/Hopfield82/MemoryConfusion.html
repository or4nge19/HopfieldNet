    source
    def Hopfield82.StorageCapacity {U : Type} [Fintype U] :
    ℝ

    The StorageCapacity of a Hopfield network is the maximum number of patterns that can be stored and reliably retrieved. The paper suggests this is around 0.15N.

    From the paper (p.2556): "About 0.15 N states can be simultaneously remembered before error in recall is severe."

    Equations
    • Hopfield82.StorageCapacity = 0.15 * ↑(Fintype.card U)

        ℝ

        The PatternRetrievalError function computes the probability of an error in pattern retrieval for a network storing m patterns. This increases as m approaches and exceeds 0.15N.

        This corresponds to the error probability P discussed in Equation 10 of the paper: P = ∫_σ^∞ (1/√(2π)) e^(-x²/2) dx = (1/2)(1 - erf(σ/√2))

        where σ = N/(2√(nN)) and n is the number of patterns.
